  在日常开发工作中，慎用“overflow:hidden;”清除浮动，对于父级元素内的绝对定位或者相对定位可能会起到遮挡作用。

单伪元素标签法：
``` css
    .clearfix:after{
        content:"";
        height:0;
        line-height:0;
        display:block;
        visibility:hidden;
        clear:both;
    }
    .clearfix:after{
        *zoom:1;        /*针对IE6/IE7*/
    }
```
双伪元素标签法：
``` css
    .clearfix :before,.clearfix:after{
        content: “”;
        display: table;
    }
    .clearfix:after{
        clear:both;
    }
    .clearfix {
        zoom: 1;/兼容IE/ 
    }
```
